Class Homework 3: Sketch and present 3 ideas for Locomotion in VR

3D Maneuver Gear My first locomotion technique is based on the three dimensional maneuver gear from the famous manga named Attack on Titan (Shingeki no Kyojin). In the manga the characters are using grappling hooks to move around buildings or trees. Each character has one controller in each hand, a gas tank and a grappling hook launcher on each side, and a gas evacuation system in the back. Thanks to the controllers they can send a grappling hook on a surface, pull back the grappling hook, pull on the rope fixed to the grappling hook in order to be closer to the surface they are hooked to.
